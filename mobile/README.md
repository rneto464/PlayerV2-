# Mobile App Version

This directory contains the source code for the mobile version of the PlayerV2 application.

## Getting Started

Currently, this folder is a placeholder. You can initialize a new mobile project here using frameworks like React Native (Expo), Flutter, or Ionic.

### Recommended Setup (React Native with Expo)

To initialize an Expo project here, run:

```bash
npx create-expo-app@latest .
```

Make sure you have Node.js installed.
